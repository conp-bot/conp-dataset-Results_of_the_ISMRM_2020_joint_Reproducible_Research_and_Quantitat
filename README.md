# Results of the ISMRM 2020 joint Reproducible Research & Quantitative MR study groups reproducibility challenge on phantom and human brain T<sub>1</sub> mapping

## Description
This dataset includes both raw data and processed T1 maps obtained from the 2020 challenge on inversion recovery T1 mapping organized by the International Society in Magnetic Resonance in Medicine (ISMRM) Reproducible Research Study Group (RRSG). For a comprehensive overview of the data distribution submitted for this challenge, please visit https://rrsg2020.db.neurolibre.org. It&#39;s important to note that this dataset exclusively comprises ISMRM-NIST system phantom data.
